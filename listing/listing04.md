Что выведет программа? Объяснить вывод программы.

```
package main

func main() {
    ch := make(chan int)
    go func() {
        for i := 0; i < 10; i++ {
            ch <- i
        }
    }()

    for n := range ch {
        println(n)
    }
}

```

Ответ

```
Программа выведет числа от 0 до 9 и выдасат ошибку deadlock
Числа записываются в канал и горутина останавливается пока в цикле for не считается значение
но когда все 10 цифр запишутся канал не закрывается и цикл for все еще ждет что в него придет значение, однако значение никогда не придет.

p.s.
	чтобы исправить вывод deadlock необходимо закрыть канал в горутине после цикла записи чисел
```
