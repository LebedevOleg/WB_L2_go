Что выведет программа? Объяснить вывод программы.

```
package main

import (
    "fmt"
)

func test() (x int) {
    defer func() {
        x++
    }()
    x = 1
    return
}


func anotherTest() int {
    var x int
    defer func() {
        x++
    }()
    x = 1
    return x
}


func main() {
    fmt.Println(test())
    fmt.Println(anotherTest())
}

```

Ответ

```
Программа выведет 2 1
defer выполняется после завершения функции в порядке LIFO тоесть последний зашел первый вышел
разница между функциями в том, что return оформлен по разному
в случае test() переменная, которую мы возвращаем инициализированна изначально и когда происходит return сначала выполняется defer и затем возвращается результат в виде х
в случае anotherTest() явно указывается какая переменная возвращается в строке return, и когда выполняется return сначала возвращается переменная и затем происходит defer
тоесть возврат переменной x в test() происходит как еще один defer созданный раньше всех
```
