Что выведет программа? Объяснить вывод программы.

```
package main

import (
  "fmt"
)

func main() {
  var s = []string{"1", "2", "3"}
  modifySlice(s)
  fmt.Println(s)
}

func modifySlice(i []string) {
  i[0] = "3"
  i = append(i, "4")
  i[1] = "5"
  i = append(i, "6")
}

```

Ответ

```
Программа выведет слайс [3,2,3]

Когда слайс передается в функции то копируется сам объект слайса внутри которого есть ссылка на текущий массив
Если в функции происходят изменения этого слайса без изменений его размера (тоесть без изменения ссылки на массив внутри слайса), то изменения отразятся в потоке где вызывалась функция
в данном случае мы меняем 0 элемент слайса и затем делаем append который возвращает новый слайс, а поскольку мы передавали копию слайса, то слайс в функции станет другим и дальнейшие действия никак не отразятся на слайсе в main

```
